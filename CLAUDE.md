# CLAUDE.md — Stressed Skin Calculator

BS 5950-9 Stressed Skin Diaphragm Calculator.
Бесплатный онлайн инструмент для конструкторов.

- GitHub Pages для хостинга

- Shear flexibility
- Shear strength
- Diaphragm stiffness
